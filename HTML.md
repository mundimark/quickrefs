# HTML <=>  Markdown Reference


HTML                            | Notes                
------------------------------- | --------------------- 
`<h1>...</h1>` - `<h6>...</h6>` | Heading 1-6
`<p>...</p>`                    | Paragraph
`<ul>(<li>...</li>)+</ul>`      | Unordered List
`<hr>`                          | Horizontal Rule
`<img src="..." alt="...">`     | Image (with Alternative Text)



## `<h1>...</h1>` - `<h6>...</h6>` - Heading 1-6

```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

-or-

Heading 1
=========

Heading 2
---------
```


## `<p>...</p>` - Paragraph

```
Paragraph

Another Paragraph
```

## `<ul>(<li>...</li>)+</ul>` - Unordered List

```
- Item 1
- Item 2
- Item 3

-or-

* Item 1
* Item 2
* item 3

-or-

+ Item 1
+ Item 2
+ Item 3
```

## `<hr>` - Horizontal Rule

```
---       

-or-   

* * *

-or-

- - -
```

## `<img src="..." alt="...">` - Image (with Alternative Text)

```
![text](http://example.com/logo.png)
![text](logo.png)                       -- use relative path
![](logo.png)                           -- alternative text optional e.g. leave empty
```
